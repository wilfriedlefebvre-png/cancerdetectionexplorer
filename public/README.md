# Image Setup

## Background Image
To add the laboratory background image:

1. Place your laboratory background image in this `public` folder
2. Name it `lab-background.jpg` (or update the filename in `app/page.tsx` line 1903)
3. Supported formats: `.jpg`, `.jpeg`, `.png`, `.webp`

The image will automatically be used as the background for the entire application.

## Human Body Anatomical Image
To add the anatomical human body illustration:

1. Place your anatomical human body image in this `public` folder
2. Name it `human-body-anatomical.png` (or update the filename in `app/page.tsx` line 1794)
3. Supported formats: `.png`, `.jpg`, `.jpeg`, `.webp`
4. Recommended: Front-facing anatomical illustration showing skeleton, muscles, and organs

The image will be displayed in the body panel with clickable overlay regions for different body parts.
