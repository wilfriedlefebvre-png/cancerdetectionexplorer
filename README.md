# Body Condition Explorer

An educational medical condition explorer built with Next.js 13+ and React. This application allows users to explore various medical conditions, their symptoms, severity stages, and associated body regions.

## Features

- **Interactive Body Map**: Click on different body regions to filter conditions
- **Search Functionality**: Search conditions by name, symptoms, or description
- **System Filtering**: Filter conditions by body systems
- **Detailed Information**: View symptoms, severity stages, emergency signs, and untreated risks for each condition
- **Educational Purpose**: Designed for educational purposes only - not medical advice

## Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser

### Build for Production

```bash
npm run build
npm start
```
